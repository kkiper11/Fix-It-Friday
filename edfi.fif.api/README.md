# EdFi.FIF.API

## Description

GraphQL-based API for supporting the Ed-Fi Fix-it-Friday solution.

## Installation

```bash
yarn install
```

## Running the app

```bash
# development
$ yarn start

# Open http://localhost:3000/ in a browser to verify

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## TeamCity

Additional commands for TeamCity support

```bash
yarn test:ci
yarn lint:ci
```
